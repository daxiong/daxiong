MAC把socks5转为http代理

2018-03-09    原创      MAC

工作中经常遇到使用了socks5的隧道远程操作的情况，不过很多工具是不支持socks5代理的，比如终端。但是很多工具是支持http代理的，那我们就需要一个工具把socks5转为http proxy。

MAC上的polipo玩法

比较简单的做法是使用 polipo

$ brew install polipo
$ sudo cp /opt/local/etc/polipo/config.sample /opt/local/etc/polipo/config

内容如下：

# Mac: /opt/local/etc/polipo/config
# Ubuntu: /etc/polipo/config
socksParentProxy = "127.0.0.1:1080"
socksProxyType = socks5
proxyAddress = "::0"        # both IPv4 and IPv6
# or IPv4 only
# proxyAddress = "0.0.0.0"
proxyPort = 8123

启动polipo

$ polipo -c /opt/local/etc/polipo/config

那么这样，我们就实现了一个127.0.0.1:8123的http代理了。

终端使用代理的玩法

由于mac的终端是可以通过设置http_proxy环境变量来实现代理的设置的，方法如下：

$ export http_proxy=http://127.0.0.1:8123
$
$ #把以上代理注册为快速命令
$ alias hp="http_proxy=http://127.0.0.1:8123”
$
$ #使用代理来执行命令
$ hp curl ip.gs
$
$ # 看看是不是已经成功走了代理了?
